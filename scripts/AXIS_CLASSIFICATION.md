## 三、关键发现

### 3.1 语义轴 vs 残差轴

| 特征 | 语义轴（64维） | 残差轴（960维） |
|------|---------------|----------------|
| 来源 | 反义词差值 | 剩余嵌入空间 |
| 内容 | 语义对立关系 | 语言/形式差异 |
| 可解释性 | 高（明确的语义维度） | 中（26.4% 已识别） |
| 主要功能 | 捕获语义"相反" | 捕获形式"相似" |

### 3.2 反义词的本质

**核心发现**：反义词在向量空间中**不是方向相反**，而是：
- 余弦相似度均值：0.827（高度相似）
- 所有已知反义词对相似度 >= 0.6

**解释**：反义词共享语法上下文（如"好的选择"、"坏的选择"），因此向量相似度高。语义对立体现在**差值方向**，而非绝对位置。

### 3.3 残差空间的功能

残差空间主要捕获：
1. **语言区分**（~27%）：不同语言的词汇分离
2. **形式特征**（~5%）：大小写、词长等
3. **形式相似性**：词形变化、同语言同义词
4. **噪音/混合**（~73%）：未识别的细粒度特征

## 四、应用指南

### 4.1 发现反义词

使用**语义轴投影法**：
1. 从已知反义词学习语义轴
2. 将词汇投影到语义轴
3. 找投影值符号相反且差距大的词对

```bash
python opposites.py --method projection --top-k 20
```

### 4.2 评估反义词质量

使用 `--method evaluate`：
```bash
python opposites.py --method evaluate
```

输出：
- 投影分离度分布
- 弱分离词对列表（需清理）
- 强分离词对列表（高质量）

### 4.3 分析残差空间

使用 `--method residual`：
```bash
python opposites.py --method residual --vocab-size 5000
```

输出：
- 语言轴分类
- 形式相似轴
- 跨语言翻译对

## 五、待解决问题

1. **73.6% 未识别残差轴**：需要更精细的分类规则
2. **语言检测准确性**：当前规则对部分语言识别不准
3. **跨语言语义轴**：残差空间中跨语言翻译对较少
4. **词性/语法轴**：尚未系统识别词性、时态等语法差异轴
