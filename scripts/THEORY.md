# 概念几何分析实验结果

本文档记录语义空间分析的实验结果，验证事物原理六大定律。

## 验证方法

### 正向词对验证

理论假设：同一原理中的正向概念应在原始空间中保持相关。

验证标准：`cosine(w1, w2) > 0.5`

### 对立词对验证

理论假设：对立概念应在语义空间中方向相反。

验证标准：`cosine(w1_sem, w2_sem) < -0.3`

### 定律验证状态

| 状态 | 验证率 | 说明 |
|------|--------|------|
| verified | ≥ 75% | 理论得到强支持 |
| partial | 50% - 74% | 部分支持，需进一步验证 |
| not supported | < 50% | 未获得支持 |

## 核心发现

### 语法空间分析有效性

语法空间成功分离 8/18 反义词对（相似度 < -0.3）：

| 词对 | 语法空间相似度 |
|------|---------------|
| 意义-无意义 | -0.98 |
| 正确-错误 | -0.86 |
| 效率-低效 | -0.84 |
| 真实-虚假 | -0.66 |
| 无知-知识 | -0.67 |
| 自由-奴役 | -0.49 |
| 进化-退化 | -0.35 |

**结论**：语法空间分析能分离部分反义词，但仍有 10/18 词对未分离或正交。

## 改进建议

### 方法学改进

1. **多词对平均**：提高概念轴稳定性
2. **层次选择**：中间层可能更"语义"
3. **扩大词表**：当前样本量不足，影响 PCA 稳定性

### 阈值调整

当前阈值（正向 0.5，反向 -0.3）可能过于严格。可考虑：
- 根据具体领域调整阈值
- 使用统计显著性检验替代固定阈值

### 实验设计

1. **对比不同模型**：验证模型依赖性
2. **对比不同层**：找到最佳语义表示层
3. **扩大词汇量**：提高子空间分解可靠性
