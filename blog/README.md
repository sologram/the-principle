您好，想问下最近遇到的关于哲学的一些困扰。我最近偶然看到了尼采关于道德的说法，包括道德是被弱者强加的对于强者的束缚，以及进一步了解的一些元伦理的观念对于道德的讨论。

这让我有些不安，比如在和一个哲学系学生讨论的过程中，其说元伦理学中有否认道德存在的观点，比如还原主义到最后道德涉及到的是利益，因此没有必要存在，但这仍然不否认其作为一种共识或者观念的存在来提供便利，就像法律能够被进一步还原，但仍然存在，不是吗？

而当我说进化心理学上的人类利他行为与团体内互助对于道德的发展的形成作用，建构主义的道德理性协商，以及互惠利他等等被证实的道德形成原因与内在理由作为可以信任这些基本道德（包括不随意伤害他人，互相尊重）对于大众的意义的证明时，他就说这些不是来自于知识的证明而只是信仰，因为这些都是不是完备的且都有缺陷，因此不是知识而只是信念，也没法佐证道德的存在。这就让人难以接受，因为这些都是学术界理论结合实践形成的知识，也是一般定义下的知识，不是吗？

如果任何知识都需要完全解释一切并且没有丝毫的偏差，那就不是知识而是绝对真理了。因此，道德在一个角度上仍然不是存在于我们的共识中，来自于如之前所论证的形成过程，有利于彼此的利益吗？

之前的这些论调让我有些困扰，因此如果可以的话，还是想要咨询下您的看法。对我来说，我一直奉行的都是尽量不首先损害他人利益的原则，并且关心他人感受，这不但让我与其他人的互动中感到愉快，也保护了彼此的利益。而这些基本的道德框架，包括不伤害、互相尊重等仍然是在这些知识的论证基础上以及实践的可行性上可以信任的，对吧？

我们通常的证明也仍然是可行的吧？
